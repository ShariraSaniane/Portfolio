VACATIM

VACATIM is Platform that can make it easier for users to find tourist areas in the province of East Java

  • Technology: Figma, Canva
  • Client: Society

Featured Project


Background

Indonesia is a country rich in natural beauty and cultural diversity, with a society consisting of various tribes, religions and races. This diversity is a basic strength in tourism development. East Java Province has great tourism potential, but there are still many areas that are not widely known. Therefore, a platform is needed to make it easier for tourists to find interesting tourist destinations. This application not only makes it easier to find tourist locations, but also provides opportunities for investors to support tourism development in East Java, which in turn can increase regional income.

Solution

The Vacatim (Vacation in East Java) application uses a mobile platform with the Android operating system. This platform was chosen because it has properties that are very easy to use, especially based on data that we quote from Statista (July 2020), the number of Indonesians using cellphones in 2021 has more than 70%. This is a very big opportunity for us to introduce a mobile-based application that can make it easier for people to find tourist attractions in East Java, namely an application called Vacatim (vacation In East Java).
